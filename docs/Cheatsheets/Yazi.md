# Yazi Cheatsheet

| Key        | Action                                                                      | Notes                  |
| ---------- | --------------------------------------------------------------------------- | ---------------------- |
| `s`        | Fuzzy find files via `fd` + `fzf` in the current directory subtree          | Overridden in dotfiles |
| `S`        | Live search file contents via `rg` + `fzf` in the current directory subtree | Overridden in dotfiles |
| `z`        | Fuzzy find files/directories via `fzf`                                      |                        |
| `Z`        | Jump to directories via `zoxide`                                            |                        |
| `<space>`  | Select/unselect highlighted file                                            |                        |
| `r`        | Rename  (selected/highlighted files)                                        |                        |
| `x` -> `p` | Cut -> Paste (selected/highlighted files)                                   |                        |
