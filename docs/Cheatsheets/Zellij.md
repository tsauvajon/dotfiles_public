# Zellij Cheatsheet

## Pane mode

| Key      | Action            | Notes |
| -------- | ----------------- | ----- |
| `ctrl-p` | Enter "Pane" mode |       |
| d        | Split down        |       |
| n        | Split right       |       |
| x        | Close             |       |


## Move Pane mode

| Key                                     | Action                                         | Notes |
| --------------------------------------- | ---------------------------------------------- | ----- |
| `ctrl-h`                                | Enter "Move pane" mode                         |       |
| h / ←  <br>j / ↓   <br>k / ↑  <br>l / → | mov left<br>move down<br>move up<br>move right |       |
|                                         |                                                |       |
|                                         |                                                |       |
`CSI 2031` -> automatic switch between dark and light mode
